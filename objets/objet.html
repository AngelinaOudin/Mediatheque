GoldenEye 007

Type : Jeu vidéo
Description :

GoldenEye 007 est un jeu de tir à la première personne, sorti sur Nintendo 64 et basé sur le film de James Bond : GoldenEye
Votre nom est Bond, James Bond.
Vous êtes désigné pour couvrir les opérations en relation avec le satellite armé Golden Eye. Votre mission débutera dans le complexe du barrage de Byelomorye en URSS.
